# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

1. Student — A college student or learner actively taking courses and completing assignments
2. Returning User — A student who has been using Mitra over time and has accumulated notes, materials, and assignment history
3. Exam Prepper — A student specifically focused on reviewing and preparing for an upcoming exam

## Draft User Stories

1. As a student, I want to create a new course so that I can organize all my notes, materials, and assignments by class.

2. As a student, I want to add notes linked to a specific course so that my study material stays organized and easy to find.

3. As a student, I want to create assignments with due dates and course associations so that I can track what I need to work on and when.

4. As a student, I want to edit and delete my notes so that I can keep my knowledge base accurate and up to date.

5. As a student, I want to upload course materials (like syllabi and slides) and link them to a course so that everything I need for a class is in one place.

6. As a student, I want to open an assignment and see all the relevant notes and materials automatically surfaced so that I don't waste time searching for what I need.

7. As a student, I want to quickly add a note or assignment from a modal without leaving the page I'm on so that I can capture ideas without losing my context.

8. As a student, I want to filter my notes by course, tag, or date so that I can quickly find specific material.

9. As a student, I want to sort my assignments by due date or completion status so that I can prioritize my workload.

10. As an exam prepper, I want to enter an exam topic and see a compiled review page of all related notes, materials, and past assignments so that I can study efficiently.

11. As an exam prepper, I want Mitra to suggest study strategies and methods based on the type of content I'm reviewing so that I can study more effectively.

12. As a returning user, I want the app to remember my past assignments and study history so that its recommendations get smarter and more relevant over time.

13. As a returning user, I want to see a dashboard overview of all my courses with progress indicators so that I know where I stand at a glance.

14. As a student, I want to receive a personalized study plan based on my upcoming deadlines so that I can manage my time and preparation across multiple classes.

15. As a student, I want form inputs to be validated (e.g., assignment must have a future due date, notes must have a title) so that my data stays clean and useful.
